Performance Comparison of React, Angular, Knockout and Setu

How to run?

  • Click Run on the right of each framework. It would run a test that produces 1000 elements from a loop/repeat template.
  • The time taken by the test replaces the text Run but you can click it again to run the test again.
  • An average of at least 10 runs per framework gives a better feel of the time taken by it in general. If you run the tests in quick succession you might observe spikes.

Credits: this performance test is adapted from this article with a demo here, which originally compares React, Angular and Knockout. The source code has been modified to include Setu tests.
